# Network Scanning using Nmap

## Objective

To scan a system and network using Nmap and identify open ports and services.

## Tools Used

* Nmap

## Key Concepts

* Port scanning
* Service detection
* Network exposure

## Steps Performed

* Scanned local system (127.0.0.1)
* Scanned system using IP address
* Performed service detection (-sV)
* Conducted aggressive scan (-A)
* Scanned network gateway (router)

## Results

* Identified open ports and services
* Observed system and router-level exposure
